# Object Detection Using YOLOv9c

## Setup

Install below libraries and packages:
1. flask
2. opencv-python
3. numpy
4. ultralytics
5. pillow
6. werkzeug

```
pip install flask opencv-python numpy ultralytics pillow werkzeug
```
